### 1. 选股池

沪深300 + 中证500 成分股（需Tushare获取成分股列表）

## 全局参数

| 参数 | 值 | 说明 |
|---|---|---|
| 初始资金 | 1亿元 | initial_cash = 100,000,000 |
| 最大持仓数 | 200只 | max_positions = 200 |
| 单只仓位上限 | 50万元 | position_limit = 初始资金 / 最大持仓数 |
| ATR周期 | 20日 | atr_period = 20 |
| ATR乘数 | 1.5 | atr_multiplier = 1.5 |
| 止盈下限 | 3% | take_profit_min_pct = 0.03 |
| 止盈上限 | 12% | take_profit_max_pct = 0.12 |
| DEA回溯天数 | 5日 | dea_lookback_days = 5 |
| 加仓阻断阈值 | 1% | max_bullish_candle_pct ≤ 0.01 |
| 佣金率 | 0.03% | 买卖均收 |
| 印花税 | 0.1% | 仅卖出收 |

## 2. 每个交易日尾盘操作

1. 检测卖出信号

   1. 止盈规则
      - 止盈阈值为 ATR 动态计算：tp1 = clamp(1.5 × ATR%, 3%, 12%)；tp2 = tp1 × 2
      - **第1次止盈**：持仓均价浮盈 ≥ tp1 → 尾盘卖出初始建仓量的1/3
      - **第2次止盈**：持仓均价浮盈 ≥ tp2 → 尾盘再卖初始建仓量的1/3
      - （止盈2次后剩余约1/3，等待MA25清仓或MA60止损）

   2. 止损规则（全程有效）
      - 收盘价跌破MA60（close < MA60）→ 标记进入"观察期"
      - **次日**：若收盘仍 < MA60 → 尾盘清仓
      - 若次日收盘 ≥ MA60 → 解除观察期，继续持有

   3. 清仓规则（止盈2次后激活，与止损并行）
      - 止盈已完成2次后，额外激活：
        - 收盘下穿MA25（close < MA25）→ 进入"MA25观察期"
        - 次日收盘仍 < MA25 → 尾盘清仓
        - 若次日收盘 ≥ MA25 → 解除观察期

   - MA60止损在止盈2次后仍然有效（两个条件哪个先触发先执行）

2. 扫描入场信号，保存数据

   1. 今日为阴线（close < prevClose）
   2. DEA刚上穿0轴（当日DEA > 0，过去5日内至少有1日DEA < 0）
   3. 收盘价 > MA60

## 3. 每个交易日早盘操作

1. 买入操作
   - 用昨日扫描的股票，以开盘价市价单买入
   - 买入量按距MA60距离分两档：
     - 距MA60 ≤ 1%（close - MA60）/MA60 ≤ 0.01：买入满仓（50万元），且禁止后续加仓
     - 距MA60 > 1%：买入1/3仓（约17万元），后续可加仓最多2次

2. 加仓操作（每笔最多加仓2次）
   - 触发条件（与入场条件相同）：
     1. 今日为阴线（close < prevClose）
     2. 当日DEA > 0，过去5日内至少有1日DEA < 0
     3. 收盘价 > MA60
   - 额外阻断条件：持仓期内单日阳线涨幅最大值 > 1% → 禁止加仓
   - 每次加仓量：1/3仓（约17万元）
